üéÆ Media Bias Sorting Game

Interactive drag-and-drop game where students categorize news sources by political bias (Left, Center, Right). Features a real-time leaderboard tracking completion times and authentication via GitHub credentials.

How We Built It
localStorage API Drag & Drop API Flask Backend SQLite Database GitHub OAuth

Persistent game state using localStorage with session recovery, server-side leaderboard using POST /api/media/score, and click tracking for popular prompts.

ü§ñ AI-Powered Help Chat

Real-time assistance chatbot providing hints or detailed information about news sources and media literacy. Features smart prompts ranked by popularity that update live every 5 seconds.

How We Built It
Anthropic Claude API WebSocket-style Updates Flask REST API JSON Storage

Integrated Claude Sonnet via fetch('/api/chat') with mode selection (hint/info). Live prompt ranking using setInterval() and server-side click counting at /api/prompts/clicks.

üìù AI Thesis Generator

Generate multiple thesis statements with strength scores (1-10), supporting arguments, and counterarguments. Analyzes topic, position, and optional supporting points to create 3-5 unique thesis variations.

How We Built It
Claude AI Dynamic Forms JSON Parsing Copy-to-Clipboard API

Backend endpoint POST /api/thesis/generate sends structured prompts to Claude, returning JSON with thesis statements, strength ratings, and recommendations. Frontend displays with copy functionality.

üìö Citation Generator

Automatically generate APA, MLA (9th ed.), and Chicago author-date citations. Features metadata scraping from URLs and a persistent "Works Cited" list with 100+ citation storage capacity.

How We Built It
DOMParser API Meta Tag Scraping CORS Proxy localStorage Custom Algorithms

Implemented parseMetadataFromHtml() to extract Open Graph tags, publication dates, and author info. Backend proxy at /api/media/fetch_meta bypasses CORS. Citations formatted via custom functions (fmtAPA, fmtMLA9, fmtChicago).

üìä Performance Reflection & Analytics

Students rate their understanding (1-5) and receive personalized resource recommendations. Admins view aggregated statistics and individual performance histories with role-based access control.

How We Built It
Session Management Role-Based Auth SQLite Queries DataTables.js Real-time Stats

Backend endpoints: POST /api/performance/submit, GET /api/performance/stats. Frontend checks user.role === 'Admin' to display management dashboard with edit/delete capabilities via PUT and DELETE requests.

üß† AI Bias Profile Analysis

Comprehensive personality assessment using Google Gemini AI to analyze student activity patterns, media consumption habits, and learning strengths/weaknesses across the entire module.

How We Built It
Google Gemini Data Aggregation Structured JSON Multi-source Analysis

Collects data from all activities (game prompts, chat questions, thesis topics, citations, performance ratings) via collectUserData(). Sends to POST /api/analyze-bias/:uid which combines frontend + backend data, prompts Gemini, and returns structured insights.

üéØ Admin Management Dashboard

Complete oversight panel for teachers and administrators. View user profiles, track GitHub commits (30-day rolling), manage performance ratings, and monitor media game leaderboard with CRUD operations on all data.

How We Built It
DataTables GitHub API RESTful CRUD Session Auth Batch Processing

Three-tab interface (Users, Performance, Media) with tabbed navigation. Fetches GitHub commits via POST /api/analytics/commits/:uid with rate limiting (15 users per batch, 4s delay). Edit/delete actions use PUT and DELETE methods with confirmation dialogs.
